MemeLib

Provides methods of scanning, connection, and commands to JINS MEME
Responses from JINS MEME are provided via each delegate method of MemeLibDelegate.

NOTE

You need to set up your own App ID / App Secret for MemeLib via setAppClientID.
Without App ID / App Secret, MemeLib won't work at all.
